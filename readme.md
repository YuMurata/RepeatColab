# RepeatColab

RepeatColabは[GoogleColaboratory](https://colab.research.google.com/notebooks/welcome.ipynb)の時間制限問題を無視して1日中動かし続けるプロジェクトです。

## Requirement
* Tool
    * python3.8
    * Chrome
    * ChromeDriver
* module
    * selenium
    * pydrive
